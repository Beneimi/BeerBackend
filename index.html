API usage

Method Path Response Body
GET beers/ List of all beers -
GET beers/name={name} Beer with the given name -
GET beers/brewery={brewery} Beers with the given brewery -
GET beers/id={id} Beer with the given id -
GET beers/ratinghigher={rating} Beers with rating higher than the given value -
GET beers/type={type} Beers with the given type -
POST beers/add Beers with the given type Beer formed as JSON
